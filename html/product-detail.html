Magenta Rose Font

We’re proud to introduce Magenta Rose Font

Magenta Rose – Hand lettering script font with an elegant feel. Comes with upper and lowercase characters, punctuation, numerals, and supports international languages. Also try the characters { } [ ] and some standard ligatures & alternates

Magenta Rose perfectly for magazines, books, wedding invitations, product labels, print, branding, social media post, etc
Every letter was carefully crafted and connections are smooth.

To enable the OpenType Stylistic alternates, you need a program that supports OpenType features such as Adobe Illustrator CS, Adobe Indesign & CorelDraw X6-X7.
Guides to access all alternates glyphs : http://adobe.ly/1m1fn4Y

Additional ways to access alternates, using Character Map (Windows), Nexus Font (Windows), Font Book (Mac) or a software program such as PopChar (for Windows and Mac).
